# 🏥 Hospital Triage System (Python)

Pequeño proyecto en Python que simula un sistema básico de triaje hospitalario utilizando programación orientada a objetos.

Permite clasificar automáticamente la prioridad asistencial del paciente y asignar su destino según parámetros clínicos definidos.

---

## 📌 Funcionalidades

* Definición de la clase `Paciente`
* Clasificación automática de prioridad:

  * 🔴 Roja
  * 🟠 Naranja
  * 🟡 Amarilla
  * 🟢 Verde
* Asignación de destino:

  * Box de aislamiento
  * Sala de espera general
* Procesamiento de múltiples pacientes mediante listas
* Uso de operadores lógicos (`and`, `or`, `not`)
* Separación del código en módulos (`paciente.py` y `hospital.py`)

---

## 🧠 Tecnologías utilizadas

* Python 3
* Programación orientada a objetos (POO)
* Estructuras condicionales (`if / elif / else`)
* Bucles `for`
* Listas de objetos

---

hospital-triaje-python/

## ▶️ Ejecución

Ejecutar el archivo principal:

```
python hospital.py
```

El programa procesará los pacientes definidos y mostrará su prioridad asistencial y destino.
